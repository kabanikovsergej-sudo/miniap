# NIGHTCOREX — Telegram Mini App

Мобильная версия NightcoreX для запуска внутри Telegram.

## Что осталось в Mini App

- Главная страница и быстрый доступ к инструментам
- BP-калькулятор: уровень, XP, дата или количество дней до конца BP, темп XP и бонусы
- Задания BP
- Таймеры
- Готовка, EMS, алкоголь, Таро, Magic Ball
- Профиль, уведомления и админ-разделы по правам доступа

## Что намеренно убрано

- Обычный калькулятор доходов/расходов
- Аренда
- Отдельная страница Menu
- Десктопный Workspace-переключатель

Старые ссылки `/calculator`, `/rentals` и `/menu` безопасно возвращают на главную, но сами разделы не попадают в production-сборку.

## Локальный запуск

```powershell
npm install
npm --prefix backend install
npm run dev:all
```

Для настоящего входа открой приложение внутри Telegram. Настройка `.env` и деплоя описана в `ЗАПУСК_TG_MINI_APP.md`.
