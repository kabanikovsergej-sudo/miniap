# NIGHTCOREX Telegram Mini App v4

Исправлено:
- BP полностью заменён на стабильный мобильный раздел без старого тяжёлого калькулятора.
- Аренда полностью заменена на мобильные карточки вместо таблицы, которая ломалась на телефоне.
- Добавлены нормальные bottom-sheet окна для добавления объекта и запуска аренды.
- Все экраны рассчитаны на телефон: прокрутка идёт через общий контейнер Mini App, без внутренних заблокированных скроллов.
- Данные BP и аренды сохраняются в localStorage устройства.

Запуск:
```powershell
npm install --registry=https://registry.npmjs.org/
npm run dev
```
