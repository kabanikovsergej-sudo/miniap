# NIGHTCOREX Telegram Mini App v5

Что сделано:

- Вставлены свежие файлы дизайна, которые были загружены отдельно.
- Вернул актуальные страницы: Главная, BP, Калькулятор, Аренда, Таймеры, Готовка, EMS, Алкоголь, Таро, Magic Ball, VIP, Настройки.
- Страница «О приложении» не используется и маршрут `/about` ведёт на главную.
- Добавлены жёсткие мобильные CSS-фиксы для маленьких телефонов от 320px.
- Исправлен главный контейнер Telegram Mini App: страницы скроллятся внутри приложения, а не ломают высоту экрана.
- Длинные вкладки, включая Таро, Готовку и справочники, теперь должны нормально листаться вниз.
- Десктопные сайдбары/плавающие переключатели скрываются в Mini App, используется нижнее мобильное меню.
- Добавлен `.npmrc`, чтобы установка зависимостей шла через обычный npm registry.

Запуск:

```powershell
npm install --registry=https://registry.npmjs.org/
npm run dev
```

Если `node_modules` не удаляется, закрой VS Code и выполни:

```powershell
taskkill /F /IM node.exe
cmd /c "rmdir /s /q node_modules"
npm install --registry=https://registry.npmjs.org/
npm run dev
```
