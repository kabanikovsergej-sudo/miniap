# NIGHTCOREX Telegram Mini App v3

Что изменено:

- Исправлена мобильная оболочка под телефон: страницы больше не сжимаются по высоте.
- Добавлена нормальная вертикальная прокрутка внутри Mini App для длинных разделов.
- Исправлена вкладка «Карты Таро»: теперь длинный список карт листается вниз.
- Удалена вкладка «О приложении» из меню и роутов. Ссылка `/about` мягко возвращает на главную.
- Возвращены разделы «Калькулятор» и «Аренда».
- Старый BP-калькулятор удалён из навигации и исходников.
- Вместо BP-калькулятора раздел `/bp` теперь открывает полноценный Battle Pass со списком заданий, отметками и прогрессом.
- Нижнее меню адаптировано под телефон: Главная / BP / Калькулятор / Аренда / Ещё.
- Таймеры перенесены в меню «Ещё», чтобы нижняя панель не была сплющенной.
- Добавлена `.npmrc`, чтобы `npm install` скачивал пакеты с официального `registry.npmjs.org`, а не с внутреннего адреса.
- Удалён старый `package-lock.json`, чтобы на твоём ПК npm создал новый нормальный lock-файл.

Как запускать:

```powershell
cd "C:\Users\mobsi.off\Desktop\tg"
cmd /c "rmdir /s /q node_modules"
Remove-Item package-lock.json -Force -ErrorAction SilentlyContinue
npm install --registry=https://registry.npmjs.org/
npm run dev
```

Если Windows не даёт удалить `node_modules`:

```powershell
taskkill /F /IM node.exe
cmd /c "rmdir /s /q node_modules"
npm install --registry=https://registry.npmjs.org/
npm run dev
```
