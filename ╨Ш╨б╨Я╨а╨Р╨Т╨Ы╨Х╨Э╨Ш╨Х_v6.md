# NIGHTCOREX Mini App v6

Исправления по реальному тесту на телефоне:

- подключён Telegram WebApp SDK в `index.html`;
- исправлена проверка Telegram Mini App в `src/lib/telegram.js`;
- принудительно включена dark theme, чтобы Tailwind `dark:` классы не отдавали белые карточки;
- полностью переделана главная страница под телефон;
- кнопки `Новости` и `Поддержка` теперь видны на главной, а не прячутся за нижним меню;
- окно новостей теперь имеет нормальную кнопку закрытия и закрывается по затемнению;
- BP переписан в тёмном мобильном стиле без белых карточек;
- Аренда переписана в карточки под телефон, без таблицы и сжатого текста;
- Калькулятор переписан в мобильный тёмный вид без кривых кнопок и широких блоков;
- добавлен `react-is` в `package.json` для Render/Recharts.

После замены файлов:

```powershell
npm install --registry=https://registry.npmjs.org/
npm run build
```

Потом:

```powershell
git add .
git commit -m "Fix mobile Mini App layout v6"
git push
```
